# LeetCode Solutions Complexity Analyzer based on Gemini Flash!

## Run locally using Docker

* **Build:** `docker build -t leetcode-complexity-analyzer .`

* **Run:** `docker run -d -p 5000:5000 leetcode-complexity-analyzer`
